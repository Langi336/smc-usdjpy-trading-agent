# USD/JPY SMC Trading Agent

## 🚀 Overview
An AI-powered trading agent that analyzes USD/JPY using Smart Money Concepts (SMC) with multi-timeframe analysis.

## 📊 Features
- **Multi-Timeframe Analysis**: Analyzes 1H-2H for bias and 5M-15M for entries
- **Smart Money Concepts**: Order blocks, FVGs, liquidity zones
- **Deep Learning Models**: TFT, Bi-LSTM, CNN-ViT ensemble
- **Telegram Integration**: Real-time signals with charts
- **Risk Management**: Automated position sizing and stop-loss
